**Explainable Deep Learning Framework for Tomato Disease Detection and Severity Estimation**

Tomato diseases significantly affect crop yield and agricultural productivity. Early and accurate disease detection is essential to reduce crop losses and improve farming outcomes. This project proposes an Explainable Deep Learning Framework for Tomato Disease Detection and Severity Estimation using advanced deep learning techniques.

The system analyzes tomato leaf images to identify various diseases and estimate their severity levels. An EfficientNetB0-based deep learning model is employed for disease classification, while Explainable AI (XAI) techniques are used to provide visual explanations of the model's predictions. The framework helps users understand the affected regions of the leaf responsible for the detected disease, increasing transparency and trust in the system.

The proposed solution aims to support farmers and agricultural experts by enabling early disease diagnosis, severity assessment, and informed decision-making. The project contributes to precision agriculture by combining high prediction accuracy with model interpretability, making it suitable for real-world agricultural applications.

Key Features:
• Automated tomato leaf disease detection.
• Disease severity estimation.
• Explainable AI for transparent predictions.
• User-friendly decision support for farmers.
• Scalable framework for smart agriculture applications.

Technology Stack:
• Python
• TensorFlow / Keras
• EfficientNetB0
• LIME / Grad-CAM
• NumPy, Pandas, Matplotlib
